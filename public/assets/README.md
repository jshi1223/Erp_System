# Assets Structure

- `css/` -> shared stylesheets (`admin-style.css`, `erp-clean.css`, `login.css`, `shared.css`, `user-style.css`, `white-green.css`)
- `js/` -> shared client scripts (`auth-guard.js`, `erp-core.js`)
- `img/` -> static images (`kvsk-logo.jpg`)

Feature-specific scripts live beside each page folder, such as `public/company/company.js` and `public/accounts-payable/accounts-payable.js`.
